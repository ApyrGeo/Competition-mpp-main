# Competition Project

Project created during DPE's Lab Assignements. (Design and Programming Environments - Medii de Proiectare si Programare)

# Features

* User & Competition Management
  * Create, update, and manage users
  * Enroll users in competitions
* Multi-language Implementation
  * Same functionality implemented in C# and Java
  * Communication between same-language and cross-language instances using gRPC + Protocol Buffers
* Client–Server System
  * Concurrent server with GUI client
  * Client–Proxy pattern for communication
* REST Services
  * Implemented REST services for the functionalities
  * Implemented WS compatibility
  * Web App in React that uses the services (in ![Java version](https://github.com/ApyrGeo/Competition-mpp-java/tree/master/competition-react-client))
* Engineering features
  * Logging with log4net / log4j
  * Separation of code into modules for maintainability
* Data Persistence
  * Shared database used by all application instances
